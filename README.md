# Maple Quest 〜きのこ島の大冒険〜

メイプルストーリー風の可愛い2Dドット絵・横スクロールアクションRPG。
**スマホ縦画面のタッチ操作に最適化**された PWA 対応 Web ゲームです。

すべてのグラフィック・サウンドはコードから生成されるオリジナル素材です(外部アセット不要)。

## 遊び方

```bash
npm install
npm run dev        # 開発サーバー起動 (http://localhost:5173)
```

スマホで遊ぶ場合は同じ Wi-Fi 内から `http://<MacのIP>:5173` にアクセスするか、
ビルドして HTTPS の静的ホスティング(Vercel / Netlify / GitHub Pages 等)に `dist/` を配置してください。
HTTPS 環境では「ホーム画面に追加」でフルスクリーンのアプリとしてオフラインでも遊べます(PWA)。

```bash
npm run build      # アイコン生成 + 型チェック + 本番ビルド → dist/
npm run preview    # 本番ビルドの確認 (http://localhost:4173)
```

## ゲーム内容

| 要素 | 内容 |
|---|---|
| 操作キャラ | **剣士**(近接・高HP)と**魔法使い**(遠距離・回復)をいつでも交代 |
| ステージ1 | ヘネシスの草原 — ブルースネイル / オレンジマッシュ → ボス **マッシュモム** |
| ステージ2 | オルビスの雲海 — グリーンスライム / リボンブタ → ボス **ピンクビーン** |
| ステージ3 | 闇の神殿 — ダークバット / ゾンビマッシュ → ボス **シグナス** |

- 一定数のモンスターを討伐するとボスが出現。ボス撃破でポータルが開き次のステージへ
- レベルアップで全ステータス成長&全回復。敵は回復薬をドロップすることがある
- ボスはそれぞれ固有の攻撃パターン(ジャンププレス衝撃波 / オーブ弾幕 / 落雷&羽根弾)を持ち、HP半分以下で強化
- ステージ進行は自動セーブされ、タイトルの「つづきから」で再開可能

## 操作方法

**スマホ(縦画面)**: 画面下の仮想ボタン
左下 ◀▶ 移動 / 右下 攻撃(長押し連打)・ジャンプ / 紫ボタン スキル3種 / 赤青ボタン HP・MP回復薬 / 左上 交代

**キーボード(PC)**: ←→ 移動 / ↑・Space・Z ジャンプ / X 攻撃 / A・S・D スキル / Q・W 回復薬 / E キャラ交代

## 技術スタック

- [Phaser 3](https://phaser.io/) + TypeScript + Vite
- ドット絵: コード内のピクセルグリッド定義から Canvas テクスチャを動的生成 ([src/sprites.ts](src/sprites.ts))
- サウンド: WebAudio によるチップチューン SFX / BGM ([src/audio.ts](src/audio.ts))
- PWA: manifest + Service Worker(キャッシュファースト)+ アイコン自動生成 ([scripts/gen-icons.mjs](scripts/gen-icons.mjs))
