# 無限ボス道場

20階のボスに挑む2Dドット絵アクション。スマホ縦画面のタッチ操作に最適化した PWA 対応 Web ゲーム。

グラフィック・サウンドはすべてコードから生成するオリジナル素材（外部アセット不要）。

## 起動

```bash
npm install
npm run dev      # 開発サーバー (http://localhost:5173)
npm run build    # アイコン生成 + 型チェック + 本番ビルド → dist/
npm run preview  # 本番ビルドの確認 (http://localhost:4173)
```

`dist/` を HTTPS の静的ホスティング（GitHub Pages 等）に置くと、「ホーム画面に追加」でフルスクリーンのアプリとしてオフラインでも動作する。

## ゲーム内容

- **20階の道場**: 各階にボスが1体。撃破するとポータルが開き次の階へ。5階ごとに強敵が登場。
- **難易度**: EASY / NORMAL / HARD / 鬼モード。20階制覇で次の難易度が解放され、敵が一段階強くなる。
- **2ジョブ・5次転職**:
  - 戦士（剣士 → スピアマン → ドラゴンナイト → ダークナイト → ダークナイト・極）
  - 魔法使い（魔法使い → ウィザード → メイジ → アークメイジ → アークメイジ・極）
  - 転職は Lv10 / 30 / 60 / 100。ゲーム中はいつでも交代できる。
- **成長**: レベルは永続保存（最大 Lv999）。撃破でレベルが上がり、ステータスとスキルが強化。多段ヒットのスキルや、レベル比による命中・被ダメージ補正、クリティカル（最大60%）を実装。
- **スタンス**: 被弾時のノックバック耐性。戦士は転職ごと +20%（5次100%）、魔法使いは +10%（最大50%）。
- **無限ボス**: 1分間でどれだけ累積ダメージを与えられるかを競うモード。HPゲージは階層ごとに2倍化していくほぼ無限のボスを、現在の強さで殴り続ける。

## 操作

**スマホ（縦画面）**: 左下のスティックで移動（上に倒すとジャンプ）、右下で攻撃・スキル3種、エリクサー、ジョブ交代。

**キーボード**: ←→ 移動 / ↑・Space・Z ジャンプ / X 攻撃 / A・S・D スキル / Q エリクサー / E 交代

## 技術

- [Phaser 3](https://phaser.io/) + TypeScript + Vite
- ドット絵: ピクセルグリッド定義から Canvas テクスチャを動的生成（[src/sprites.ts](src/sprites.ts)）
- サウンド: WebAudio によるチップチューン SFX / BGM（[src/audio.ts](src/audio.ts)）
- PWA: manifest + Service Worker + アイコン自動生成（[scripts/gen-icons.mjs](scripts/gen-icons.mjs)）
- iPhone のセーフエリアに対応（上部UIをダイナミックアイランド/ノッチ下に配置）
